Interfaz de Usuario

Elementos Principales

Botones

Los botones son elementos interactivos que permiten a los usuarios realizar acciones dentro de una aplicación. Pueden tener etiquetas, íconos o imágenes que indiquen su función.

  • Botón de envío: Confirma una acción o envía un formulario.
  • Botón de cancelar: Permite anular una acción en curso.
  • Botón de menú: Despliega una lista de opciones disponibles.

Menús

Los menús organizan y presentan las opciones disponibles en una interfaz. Pueden ser desplegables, contextuales o de navegación.

  • Menú de navegación: Permite moverse entre diferentes secciones de una aplicación o sitio web.
  • Menú contextual: Aparece cuando se hace clic derecho sobre un elemento y ofrece opciones relacionadas.
  • Menú desplegable: Muestra una lista de opciones al interactuar con un botón o enlace.

Ventanas

Las ventanas son contenedores gráficos que organizan y muestran información o herramientas dentro de una aplicación.

  • Ventana principal: Es la ventana central donde se muestran los contenidos principales de la aplicación.
  • Ventana modal: Se superpone a la ventana principal y requiere una acción del usuario antes de continuar.
  • Ventana emergente: Aparece de forma independiente para mostrar información adicional o alertas.
